CSC188-Chat-Messenger
====================

VISION:
     Build innovative communication within an institution by conveying information to meet the needs for every department stations.

MISSION:
     Create a chat messenger application to provide an easy way of communication.

PRINCIPLES:
     -Always consider the request of the owner, never neglect them.
     -Always make your work as user-friendly as possible.

OBJECTIVES:
     -Within two months after project initiation, the features of the application are already implemented. 
     -Within a month after the first delivery, the users can get full access to the features of the application. 
     -Within two weeks after the second delivery, the moderator can monitored the application’s data. 
     -Within a month after project initiation, the database will be available including the user contacts.
